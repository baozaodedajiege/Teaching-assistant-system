# 面向OBE的形成性评价管理系统

## 部署
1. 用sql目录下的脚本文件初始化数据库并修改配置文件中密码
2. 解压群文件的图片文件到本地文件夹，并修改Const中的文件路径
3. 本地要安装redis，并启动redis服务，不用设置账号密码
